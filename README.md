# 📱 Buenagenda

Um aplicativo mobile desenvolvido com foco em **produtividade pessoal**, combinando funcionalidades de **lista de tarefas (to-do list)** e **agenda** em um único lugar.

## ✨ Motivação

Este projeto surgiu da necessidade de ter um aplicativo totalmente **personalizado**, já que as soluções disponíveis no mercado não atendem completamente às preferências e ao fluxo de organização desejado.

A proposta é criar uma ferramenta simples, eficiente e adaptável ao dia a dia.

---

## 🚀 Objetivo

Desenvolver um app que permita:

* Criar, editar e excluir tarefas
* Definir datas e horários para tarefas
* Visualizar tarefas em formato de lista
* Visualizar compromissos em formato de agenda/calendário
* Marcar tarefas como concluídas
* Organizar melhor a rotina pessoal

---

## 🛠️ Tecnologias

O projeto será desenvolvido utilizando:

* **React Native**
* **Expo**

Essas tecnologias permitem um desenvolvimento ágil, com fácil teste em dispositivos reais e menor complexidade inicial.
